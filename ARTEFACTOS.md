# Fundamentos de Java para Principiantes

## Resumen del Curso
Este curso introductorio está diseñado específicamente para estudiantes de secundaria de entre 12 y 17 años que no tienen experiencia previa en programación. A través de 6 sesiones breves y dinámicas, los estudiantes aprenderán lógica computacional, manejo de variables, estructuras de control, métodos y los pilares de la Programación Orientada a Objetos (POO). El objetivo principal es despertar el interés por la tecnología utilizando herramientas 100% web y un lenguaje accesible, cumpliendo con los estándares de comunicación de la carrera.

**Duración total**: ~60 minutos
**Público Objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación
**Prerrequisitos**: Ninguno
**Herramientas necesarias**: **¡Solo tu navegador web!** (Chrome, Firefox, Safari, Edge)

**📂 Repositorio de Código Fuente**: [https://github.com/tu-usuario/java-fundamentals-course-tu-equipo](https://github.com/smartdropw/java-fundamentals-course-SmartDrop)
* **Editor en línea oficial para las prácticas:** [OnlineGDB](https://www.onlinegdb.com/)

## Secuencia de la Lección

### Lección 1: ¿Qué es Java y la Programación? (~5 minutos)
- **Descripción**: Aprende qué es la programación, por qué Java es un excelente lenguaje para empezar y cómo usar el entorno en línea.
- **Enlace**: [Ver la lección](https://youtu.be/mTaE-6b19EE)
- **Conclusiones clave**: Java se usa para aplicaciones y juegos; los programas son instrucciones ordenadas. Uso de `System.out.println()`.
- **Empieza a programar**: [Abrir OnlineGDB](https://www.onlinegdb.com/) 

### Lección 2: Variables y tipos de datos (~7 minutos)
- **Descripción**: Descubre cómo las aplicaciones guardan información en la memoria usando variables y tipos de datos.
- **Enlace**: [Ver la lección](https://youtu.be/SECzxz_poMk)
- **Conclusiones clave**: Uso de `int` y `String`. Declaración estructurada y aplicación de operadores aritméticos básicos.
- **Práctica**: [Abrir OnlineGDB](https://www.onlinegdb.com/) 

### Lección 3: Estructuras de control: Decisiones y bucles (~6 minutos)
- **Descripción**: Dale el poder a tus programas de tomar decisiones lógicas con if-else y de repetir tareas de forma automática con bucles.
- **Enlace**: [Ver la lección](https://youtu.be/biEl5DHp0G4)
- **Conclusiones clave**: Estructura `if-else` con operadores (`>`, `<`, `==`) y automatización mediante el bucle `for`.
- **Práctica**: [Abrir OnlineGDB](https://www.onlinegdb.com/) 

### Lección 4: Métodos y Entrada de Datos con Scanner (~ minutos)
- **Descripción**: Organiza tu código de manera limpia usando métodos con parámetros e interactúa con el teclado del usuario.
- **Enlace**: [Ver la lección]([Colocar enlace de YouTube del Video 4 aquí])
- **Conclusiones clave**: Estructuración de métodos reutilizables. Lectura de datos por consola utilizando la clase `Scanner`.
- **Práctica**: [Abrir OnlineGDB](https://www.onlinegdb.com/)

### Lección 5: Introducción a la Programación Orientada a Objetos (POO) (~7 minutos)
- **Descripción**: Explora el verdadero corazón de Java modelando elementos del mundo real a través de clases y objetos.
- **Enlace**: [Ver la lección](https://youtu.be/5L5dnTPbg3Y?si=j0Yzo5xH5M8hLHeI)
- **Conclusiones clave**: Diferencia entre Clase (molde) y Objeto (creación). Uso de atributos, métodos y el Constructor con `new`.
- **Práctica**: [Abrir OnlineGDB](https://www.onlinegdb.com/) 

### Lección 6: Proyecto Final - Registro de Estudiantes con POO (~ minutos)
- **Descripción**: Integra todos los conceptos del curso en una aplicación real, aplicando encapsulación y solucionando errores comunes.
- **Enlace**: [Ver la lección](https://youtu.be/Cr0URZb0YHw)
- **Conclusiones clave**: Encapsulación con atributos `private` y `getters`. Uso de arreglos para almacenar objetos y control de excepciones.
- **Práctica**: [Abrir OnlineGDB](https://www.onlinegdb.com/)
